ClipMason

Native clipboard manager for macOS. Detects what you copy and offers contextual actions for text, images, colors, URLs, and files. Lives in your menu bar with global hotkey access. Built with SwiftUI. All data stays local. Supports 16 languages.

Text Transformations

22+ actions including case conversion, JSON formatting, Base64 encoding, hashing, and line sorting. Edit on the fly without leaving your workflow.

Image Editing

Resize (percentage or pixels, 25%-400%), rotate, OCR, annotate, and export. Version 1.0.2 brings fully flexible resizing for precise control.

Color Detection

Automatic extraction with format conversion: HEX, RGB, HSL, SwiftUI, CSS. Copy colors instantly from any image or screen content.

Files and URLs

Finder integration for files. URL actions for quick previews and manipulations. Everything accessible from one place.

How It Works

Step 1

Copy anything—text, image, color, URL, or file. ClipMason detects it automatically in the background.

Step 2

Click the menu bar icon or use the global hotkey to open the history.

Step 3

Select your item. Contextual actions appear based on type—transform, edit, or export instantly.

Step 4

Paste the result anywhere. Data remains local and secure.
